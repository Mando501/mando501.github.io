## Key User Interactions

### 1. Navigation
- Click navigation menu items
- Click "Generate Report" button from Tasks page

### 2. Date Selection
- Click date picker to choose start date
- Click date picker to choose end date
- Dates validate (start â‰¤ end)

### 3. Project Selection
- Click dropdown to open project list
- Check/uncheck projects (multi-select)
- Click chips to remove selected projects

### 4. File Input
- Type filename in text field
- Auto-appends .md extension if missing

### 5. Preview
- Automatically updates as selections change
- Shows task count, project count, hours, days

### 6. Actions
- Click "Cancel" to abort and return to Tasks
- Click "Generate Report" to create and download

### 7. Feedback
- Snackbar notifications (success/error)
- Error messages on form fields
- Loading spinner during generation
